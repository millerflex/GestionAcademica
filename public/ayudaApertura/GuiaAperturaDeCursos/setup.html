Instalación
Para instalar Gestión Académica UNT, siga estos pasos:
1. Descargue la última versión
Dirígete a https://www.product.com/download/ y descarga la última versión disponible (1.0).
2. Revise los requisitos del sistema y el acuerdo de licencia
Asegúrese de que su dispositivo cumpla con los requisitos recomendados para el funcionamiento estable del sistema. Además, lea y acepte el acuerdo de licencia antes de proceder con la instalación.
3. Proceso de instalación
Durante el proceso de instalación:
  • Haga doble clic en el archivo de instalación descargado para iniciar el asistente de instalación.
  • Siga las instrucciones en pantalla y seleccione la ubicación donde desea instalar Gestión Académica UNT.
  • El sistema instalará automáticamente los componentes necesarios y configurará el software.
  • Una vez completada la instalación, haga clic en "Finalizar" para completar el proceso.
    4. Verificación
    Después de la instalación, inicie el software para asegurarse de que todo esté funcionando correctamente. Si es necesario, realice una actualización del sistema o configure las preferencias iniciales.
